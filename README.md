# Data Analysis With Python

This repository contains my hands-on implementation of core data analysis concepts using Python.

## Key Highlights
- Hands-on implementation of data analysis libraries
- Real-world datasets (IPL, Student Performance, Sales)
- Focus on both data cleaning and visualization

## Topics Covered
- NumPy
- Pandas 
- Matplotlib
- Seaborn

## Projects
- IPL Team Performance Analysis using NumPy
- Sales Data Analysis using Pandas
- Student Performance Analysis using Matplotlib
- Student Performance Analysis using Seaborn
- IPL 2022 Team Performance Analysis using Seaborn

## NumPy
- Array manipulation
- Indexing & slicing
- Boolean filtering
- Aggregations
- Broadcasting
- Sorting

## Pandas
- Data Cleaning & Handling Missing Values
- Series and DataFrames
- Reading CSV and Excel files
- Data Exploration and Exercises

## Matplotlib
- Data Visualization & EDA
- Line Plots, Bar Charts, and Scatter Plots
- Histograms and Box Plots
- Heatmaps for Correlation Analysis
- Customizing Charts (Labels, Titles, Legends, and Colors)
- Subplots and Figure Layouts

## Seaborn
- Statistical Data Visualization
- Distribution Plots (Histplot, Jointplot, Pairplot, Rugplot)
- Categorical Plots (Barplot, Countplot, Boxplot, Violinplot, Stripplot, Swarmplot)
- Relational Plots (Scatterplot, Lineplot)
- Matrix Plots (Heat map, Cluster map, Pivot table heat map)
- Styling and Themes
